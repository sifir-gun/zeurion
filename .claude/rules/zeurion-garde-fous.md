# Garde-fous Zeurion

Principes non négociables, issus de l'architecture v2 et du journal de décisions. À respecter dans toute contribution au projet Zeurion.

- Phase 0 entièrement manuelle : rien ne s'automatise avant le premier euro. Ne pas proposer d'automatisation prématurée.
- Scores et décisions de capital calculés en code déterministe, jamais par un LLM. Les LLM produisent des signaux, le code décide.
- Acquisition uniquement par canaux en liste blanche. Astroturfing, faux avis, faux témoignages, contact des auteurs des posts sources : interdits (L3), sans exception.
- Aucune affirmation non sourcée sur aucune surface publique d'aucune venture.
- Le kill est plus important que le lancement : pas de complaisance envers les ventures zombies.
- Prompt caching obligatoire sur toute partie stable de contexte.

Détail complet : references/usine-business-autonome-v2.md et decisions/log.md.
