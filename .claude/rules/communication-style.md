# Style de communication

Comment écrire pour Guney, dans les réponses comme dans les livrables.

## Présentation

- Prose claire, formatage minimal. Pas de mise en forme décorative.
- Tableaux seulement quand ils apportent une vraie lisibilité (comparaisons, matrices). Jamais par défaut.

## Interdits absolus

- Jamais de tirets cadratins, dans aucune langue. Utiliser deux points, virgules ou parenthèses.
- Pas d'emojis.

## Ton interne (workspace, rapports, échanges de travail)

Direct, technique, franc. Le pushback est bienvenu : signaler les désaccords, les risques et les faiblesses d'une idée sans détour.

## Ton externe (clients Wengraf, surfaces publiques des ventures)

Professionnel, sobre. Zéro affirmation non sourcée, sans exception.
